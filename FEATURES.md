# Features

## Core Features

### AI Spend Tracking
Users can track monthly spending on various AI tools and services.

### Savings Recommendations
The system analyzes spending and recommends cheaper or better plans.

### Audit Engine
A TypeScript-based audit engine calculates:
- current spend
- recommended spend
- monthly savings
- annual savings

### Multi-Tool Support
Supported platforms:
- ChatGPT
- Claude
- Cursor
- Copilot
- Gemini
- Windsurf

### Modern SaaS UI
The application uses:
- Tailwind CSS
- shadcn/ui
- responsive layouts
- dark mode inspired design

### Responsive Design
Works across:
- desktop
- tablet
- mobile devices
